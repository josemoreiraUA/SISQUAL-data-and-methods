###### This folder keep Python files used by notebooks (in folder named notebooks) for data processing. This strategy aim to keep notebooks clean with focus on visual results.

+ `crossvalidation.py` - Has the aim to be used for implementing crossvalidation analysis;
+ `datacleaner.py` - Is the script Python used to clean data;
+ `dataexplore.py` - Is the script Python used to explore data;
+ `dataframegenerator.py` - This script has the aim to generate dataframes from csv files or from a database;
+ `detect_outliers.py` - Used to detect outliers;
+ `utils.py` - Some usefull functions - deprecated.
