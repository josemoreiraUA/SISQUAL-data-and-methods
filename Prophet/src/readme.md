###### This folder contains all sources files used in Python.

+ **data** is the folder where the Pythons scripts used to process data are kept;
+ **visualization** is the folder where the Pythons scripts used to visualize data are kept;
+ `forecast_metrics.py` is the script used to generate forecast metrics;
+ `prophet_tunning.py` aim to tune Prophet models;
+ `prophet_forecast.py` make forecasts using Prophet.
