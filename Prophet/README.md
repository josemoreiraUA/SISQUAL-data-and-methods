The structure is very simple and self explained. data is the folder to deal with data , this folder inside the src folder means Python script to deal with data. This structure need a `.env` file which must not be uploaded. This file contain users, passwords, path and other sensitive data. The `settings.py` script is responsible to read the `.env` file and the library dotenv must be imported to the project. References and details can be read at https://github.com/theskumar/python-dotenv
